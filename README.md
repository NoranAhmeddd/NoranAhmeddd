### 🚀 Featured Projects

#### 🏧 [SchnellPay ATM](https://github.com/NoranAhmeddd)
Microcontroller-based ATM interface built with **Embedded C** on the **ATmega32**, supporting both physical hardware (ETA32/ETA32mini) and **Proteus** simulation.
- Secure workflows: phone verification, OTP authentication, balance inquiries, deposits, withdrawals, transfers
- Python/Node.js bridge servers converting serial communication into HTTP requests for REST API integration
- Backend powered by **MS SQL Server** for secure account and transaction management

`Embedded C` `ATmega32` `Proteus` `Python` `Node.js` `REST APIs` `MS SQL Server`

#### 🛡️ Enterprise Network Security Infrastructure — *NTI Cybersecurity Academy Final Project*
Designed and implemented a secure enterprise network using **FortiGate Firewall** integrated with **Active Directory**, **DNS**, and web server services.
- Configured network segmentation (Internal, DMZ, External zones) and firewall policies
- Deployed **AD DS** for centralized identity management, authentication, and authorization
- Applied **Group Policy Objects (GPOs)** to enforce security and strengthen endpoint protection
- Configured traffic filtering and access control to block unauthorized external connections

`FortiGate` `Active Directory` `Windows Server` `DNS` `GPO` `Network Security`
